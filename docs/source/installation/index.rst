Installation
==============


There are different methods to install Papermerge. They differ
by amount of effort required and purpose.

.. warning::

    Starting with version 1.3.0 (which is now in development), manual installation will be simplified a lot. It will be as easy as 1. pip install papermerge 2. ./manage.py migrate 3. ./manage.py createsuperuser 4. ./manage.py runserver
